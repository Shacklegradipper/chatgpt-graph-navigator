Setup - Get Your Authentication Token

1
打开 ChatGPT 并登录

前往 https://chatgpt.com 并确保你已经登录你的账号。

2
打开开发者工具

在 ChatGPT 页面上,按下键盘上的：

  • Windows/Linux: F12 或 Ctrl + Shift + I
  • Mac: Cmd + Option + I

开发者工具会在页面底部或右侧打开。

3
切换到 Network 标签页

在开发者工具的顶部，点击 "Network" (网络) 标签页。

4
触发 API 请求

在 ChatGPT 中发送一条消息（任何内容都可以），或者刷新页面 (F5)。
这会触发 API 请求，我们可以从中获取 token。

5
查找 backend-api 请求

在 Network 标签页的请求列表中，找到任何以 backend-api 开头的请求，例如：

backend-api/conversation
backend-api/conversations
backend-api/me

点击该请求。

6
复制 Authorization Token
  1. 在右侧面板中，点击 "Headers" (请求头) 标签页
  2. 向下滚动找到 "Request Headers" (请求标头) 部分
  3. 找到 "authorization:" 这一行
  4. 它的值是 Bearer eyJhbGci... 这样的格式
  5. 只复制 "Bearer " 后面的部分（不包括 "Bearer " 这个词）

Token 通常很长（1000+ 字符），以 eyJ 开头。

7
粘贴 Token
⚠️ 安全提示：
• Token 仅存储在你的本地浏览器中，不会发送到任何第三方服务器
• Token 有效期约 24 小时，过期后需要重新获取
• 不要将 token 分享给他人
