Manual Token Setup

Follow these steps to configure your authentication token

1 Open ChatGPT and sign in

Go to https://chatgpt.com and make sure you're logged into your account.

2 Open Developer Tools

On the ChatGPT page, press:

  • Windows/Linux: F12 or Ctrl + Shift + I
  • Mac: Cmd + Option + I

The Developer Tools panel will open at the bottom or side of the page.

3 Go to the Network tab

Click the Network tab at the top of Developer Tools.

4 Trigger an API request

Send a message in ChatGPT (any content works), or refresh the page (F5). This triggers an API request we can extract the token from.

5 Find a backend-api request

In the Network tab, look for any request starting with backend-api, such as:

backend-api/conversation
backend-api/conversations
backend-api/me

Click on that request to select it.

6 Copy the Authorization token
  1. In the right panel, click the Headers tab
  2. Scroll down to the Request Headers section
  3. Find the line labeled authorization:
  4. The value looks like Bearer eyJhbGci...
  5. Copy only the part after "Bearer " (don't include the word "Bearer")

The token is usually very long (1000+ characters) and starts with eyJ.

7 Paste your token
Security Notice
  • Your token is stored locally in your browser and never sent to third-party servers
  • Tokens are valid for approximately 24 hours before needing renewal
  • Never share your token with anyone
